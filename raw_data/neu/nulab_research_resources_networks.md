The resources here include tutorials, datasets, and other useful links for creating network visualizations and performing network analysis. 
### Tools and Tutorials
Network analysis tutorials
  * _[Network Science](https://networksciencebook.com/) _– A textbook written by Northeastern Professor Albert-László Barabási.
  * [Social Network Analysis](http://www.mjdenny.com/workshops/SN_Theory_I.pdf) – Social network analysis tutorial written by Matthew Denny for the Institute for Social Science Research at UMass Amherst.
  * [Social Network Analysis basics](https://ocw.mit.edu/courses/sloan-school-of-management/15-599-workshop-in-it-collaborative-innovation-networks-fall-2011/lecture-notes/MIT15_599F11_lec04.pdf) – Social network analysis basics slideshow for the MIT Fall 2011 workshop in IT: Collaborative Innovation Networks by Johannes Putzke of the University of Cologne.


Network analysis with Python
  * [Exploring and Analyzing Network Data with Python](https://programminghistorian.org/en/lessons/exploring-and-analyzing-network-data-with-python) – A tutorial on how to perform network analysis using the Python programming language. Written by John Ladd, et al.
  * [NetworkX: Network Analysis with Python](https://www.cl.cam.ac.uk/~cm542/teaching/2011/stna-pdfs/stna-lecture11.pdf) – A tutorial from the Sunbelt 2012 Network Science Conference on how to use the NetworkX Python package. Written by Salvatore Scellato of the University of Cambridge.


Network visualization tutorials by Katya Ognyanova of Rutgers University
  * [Visualization with Gephi](http://www.kateto.net/wp-content/uploads/2016/04/Gephi%20Handout%20Sunbelt%202016.pdf) – Workshop material from the Sunbelt 2016 Network Science Conference on how to create network visualizations using [Gephi](https://gephi.org/), an open-source and free network data visualization tool.
  * [Visualization with R](http://kateto.net/network-visualization) – Workshop material from the Sunbelt 2019 Network Science Conference on how to create network visualizations using R.


### Data
Datasets
  * [Duke Network Analysis Center](https://sites.duke.edu/dnac/resources/datasets/) – A database of network dataset repositories.
  * [Networkrepository.com](http://networkrepository.com/) – A database of network data with interactive data visualization and analytical tools.
  * [SNAP Stanford Large Network Database Collection](https://snap.stanford.edu/data/) – A collection of multiple network datasets including resources from social media like Facebook, LiveJournal, and Twitter.
  * [UC Irvine Network Data Repository](https://networkdata.ics.uci.edu/) – A collection of network datasets used in previously published scholarly articles.
