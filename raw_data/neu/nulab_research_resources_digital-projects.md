# Digital Projects
An example design made with SketchUp, from Molly Nebiolo's [ forthcoming digital exhibit](https://cssh.northeastern.edu/nulab/visualizing-colonial-philadelphia/) that visualizes a portion of colonial Philadelphia.
These resources include digital project development tools for data visualizations, timelines, map-based presentations, 3D modeling, citation management, and backend website development using HTML,CSS, and JavaScript. 
### Project Planning and Development
Data Management
  * [Comparison Chart](http://subjectguides.lib.neu.edu/ld.php?content_id=46279299) – A comparison of the different citation management software packages.
  * [Northeastern Library Citation Management](http://subjectguides.lib.neu.edu/Citations) – Tutorial on citation software like Zotero, EndNote, and Mendeley.
  * [Tropy](https://tropy.org/) – An image organization platform that is useful for archival or visual culture research.


Project Development and Publication
  * [Design for Diversity](https://des4div.library.northeastern.edu/) – Resources for understanding how to create more inclusive information systems. While the site’s primary audience is cultural heritage practitioners, there are also many examples of digital projects and readings that will be useful more broadly.
  * [Digital Toolkit for Community Projects](https://cssh.northeastern.edu/nulab/program/diti/community-engagement-toolkit/) – Published by the NULab and the DITI, this Toolkit contains resources for tools that can be used to share, preserve, and celebrate local stories and heritage materials.
  * [Flask](http://flask.pocoo.org/) – A micro web framework for web applications.
  * [Zotero Library for NULab Scholars’ Publications](https://cssh.northeastern.edu/nulab/research/publications/zotero-library-for-nulab-scholars-publications/) – A Zotero library displaying and organizing the publications of NULab-affiliated scholars from 2022.


### Visualization and Publication
Data Visualization
  * [Kieran Healy’s Data Visualization](https://socviz.co/index.html#preface) – Textbook on data visualization using R.
  * [Northeastern Library Subject Guide: Data Visualization](https://subjectguides.lib.neu.edu/dataviz) – Library guide for data visualization. Curated by Kate Kryder.
  * [Northeastern Library Subject Guide: GIS ](https://subjectguides.lib.neu.edu/gis) – Library guide for Geographic Information Systems. Curated by Bahare Sanaie-Movahed.


Knight Lab 
Produced by Northwestern University’s [Knight Lab](https://knightlab.northwestern.edu/about/), this toolkit includes open-source, lightweight tools for media makers. Each resource includes tutorials and examples, as well as resources for embedding materials in a personal or project website.
  * [Juxtapose](https://juxtapose.knightlab.com/) – A tool for making side-by-side photograph comparisons with an interactive sliding threshold.
  * [Scene](https://scene.knightlab.com/) – A tool for making virtual reality presentations.
  * [Soundcite](http://soundcite.knightlab.com/) – A tool for adding inline sound clips to a text story.
  * [Storyline](http://storyline.knightlab.com/) – A tool for creating timeline-based, slideshow-like, presentations using data.
  * [StoryMap](https://storymap.knightlab.com/) – A tool for creating map-based, slideshow-like, presentations.
  * [Timeline](http://timeline.knightlab.com/) – A powerful, easy-to-use method of creating historical timelines.


3D Modeling
  * [SketchUp](https://www.sketchup.com/) – A 3D modeling toolkit for making and printing 3D models. There are free and priced versions of this product.


Website Development and Editing
  * [CodeSchool](https://www.pluralsight.com/codeschool) – Offers courses on various coding skills, costs money but there is a free trial.
  * [HTML-CSS-JS](https://html-css-js.com) – A free multifunctional web editor with instant preview, “cheat sheets,” and templates for HTML, CSS, and JavaScript.
  * [MDN WebDocs](https://developer.mozilla.org/en-US/docs/Learn) – Free in-depth tutorials on website development and various programming languages.
  * [Traversy Media](https://www.youtube.com/c/TraversyMedia/videos?view=0&sort=p&flow=grid) – A YouTube channel with detailed “crash courses” on HTML, CSS, JavaScript, and advanced programming languages.
  * [W3 Schools](https://www.w3schools.com/html/) – Completely free courses on HTML, CSS, JavaScript, Python, and more, with a built-in web editor.
  * [WebsiteSetup](https://websitesetup.org/) – A set of free guides and walkthroughs for setting up and developing WordPress sites.
